## 概要
相互合意の上、学生が病欠や怪我で休んだ際、学生(&保護者)、教師、地元医療関係者三者が情報共有できるアプリケーション。

## コンセプト
学生(&保護者)、教師、地元医療関係者をつなぐプラットフォームアプリケーション
